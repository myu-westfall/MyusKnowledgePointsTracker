# Myu's Knowledge Points Tracker

Due to community requests, I decided to convert my knowledge points tracker weakaura into an standalone Addon and add support for Midnight expansion.

**Right click** to toggle between expansions:  
![](https://media.forgecdn.net/attachments/1555/50/mkpt_expansion-gif.gif)

Slash commands

*   /mkpt - Show/hides the addon
*   /mkpt minimap - Show/hides the minimap Icon
*   /mkpt compartment - Show/hides the entry in the addon compartment
*   /mkpt scale 1.0 - Resizes the window based on the scale provided, accept values from 0.5 to 1.5

Currently Supports

*   <span style="color:#843fa1">Unique treasures</span> - Shows location and requirements of each individual treasure with TomTom integration.
*   <span style="color:#3598db">Weekly</span> - Shows which weekly Quests, Treasures and Treatises are still not completed for the week.
*   <span style="color:#2dc26b">Catch-Up&nbsp;</span>    - Tracks blizzard hidden currency which allows for extra points to be obtained with Crafting Orders, Gathering and Desenchanting.
*   <span style="color:#e67e23">Remaining</span>\- Knowledge points needed to complete the Tree.
*   <span style="color:#ff6dce">Unspent</span> - Points available to spend
